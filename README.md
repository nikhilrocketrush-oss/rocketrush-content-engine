# RocketRush Content Engine

AI-powered LinkedIn content production system for RocketRush.

## What This Is
A conversational AI system that takes a client from raw material to a
fully drafted, algorithm-scored, human-sounding LinkedIn post.

## Structure
```
rocketrush-content-engine/
├── orchestrator/     ← Custom RocketRush chatbot skill
├── skills/           ← 4 GitHub skills (install instructions)
├── clients/          ← One folder per client
├── database/         ← Master client index + post history
├── scripts/          ← Python automation scripts
└── docs/             ← Full system documentation
```

## Quick Start
1. See `docs/` for full system documentation
2. See `skills/README.md` to install the 4 skills
3. See `clients/_template/` to onboard a new client
4. See `orchestrator/SKILL.md` to activate the chatbot
